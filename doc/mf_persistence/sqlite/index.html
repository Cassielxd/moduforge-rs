基于 SQLite 的 EventStore 实现。

默认使用 WAL 日志模式与 IMMEDIATE 事务以获得低延迟写入。 采用“单连接 + 互斥”顺序写，确保追加顺序并避免写锁竞争； 读取按需准备语句，受益于 SQLite 页缓存。
