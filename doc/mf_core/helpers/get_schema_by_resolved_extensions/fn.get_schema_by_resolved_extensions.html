pub fn get_schema_by_resolved_extensions(
    extensions: &Vec<Extensions>,
) -> ForgeResult<Schema>
Expand description

根据已解析的扩展列表生成完整的 Schema

§参数

  • extensions - 扩展列表，包含节点(N)、标记(M)和扩展(E)三种类型

§返回值

  • EditorResult<Schema> - 返回编译后的 Schema 或错误

§功能说明

  1. 收集所有扩展中定义的全局属性
  2. 处理节点扩展，构建节点定义
  3. 处理标记扩展，构建标记定义
  4. 确定顶层节点名称
  5. 编译生成最终的 Schema
